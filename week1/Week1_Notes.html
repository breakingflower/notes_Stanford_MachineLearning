Gradient Descent

================

More general algo --> use GD to minimize some arbitrary cost function J(θ0,θ1)J(\theta_0, \theta_1)J(θ0​,θ1​)

  • start with some random θ0\theta_0θ0​ and θ1\theta_1θ1​
  • keep changing both to reduce JJJ
  • do this until you converge to a (local) minimum
  • if you start in a different point, you might get a different local minimum

Gradient Descent Intuition

  • Essentially taking the slope of the line that is tangent to the (cost)function.
  • If α\alphaα is too small, it takes long, if α\alphaα is too big, it might fail to converge / overshoot.

Answer to the question in the video:

  • Q: Suppose θ1\theta_1θ1​​ is at a local optimum of J(θ1)J(\theta_1)J(θ1​). What will one step of gradient descent θ1:=θ1−αδδθ1J(θ1)\theta_1:=\theta_1−\alpha\frac{\delta}{\delta\theta_1}J(\theta_1)θ1​:=θ1​−αδθ1​δ​J(θ1​) do?
  • A: Leave θ1\theta_1θ1​ unchanged

This also means that GD can converge to a local minimum even with a fixed learning rate α\alphaα. As we approach a local minimum, GD will automatically take smalelr steps. (if δ\deltaδ is steep, bigger steps, if δ\deltaδ is small, smaller steps). δ\deltaδ approaches 0 as we approach the bottom of our convex function.

Gradient Descent for Linear Regression

We want to do the update step

θj:=θj−α(δδθj)J(θ0,θ1)\genericGDθj​:=θj​−α(δθj​δ​)J(θ0​,θ1​)

on the loss function

J(θ0,θ1)=12m∑i=1m(hθ(xi)−yi)2\lrLossJ(θ0​,θ1​)=2m1​i=1∑m​(hθ​(xi​)−yi​)2

with hypothesis

hθ(x)=θ0+θ1(x)\lrHypothθ​(x)=θ0​+θ1​(x)

Figure out the partial derivative first:

δδθjJ(θ0,θ1) \deriv{j} δθj​δ​J(θ0​,θ1​)

Which simplifies to:

θ0:j=0:δδθ0J(θ0,θ1)=1m∑i=1m(hθ(x(i))−y(i)) \theta_0: j = 0 : \deriv{0} = \frac{1}{m} \sum_{i=1}^m(h_\theta(x^{(i)})-y^{(i)}) θ0​:j=0:δθ0​δ​J(θ0​,θ1​)=m1​i=1∑m​(hθ​(x(i))−y(i))
θ1:j=1:δδθ1J(θ0,θ1)=1m∑i=1m(hθ(x(i))−y(i))x(i) \theta_1: j = 1 : \deriv{1} = \frac{1}{m} \sum_{i=1}^m(h_\theta(x^{(i)})-y^{(i)}) x^{(i)} θ1​:j=1:δθ1​δ​J(θ0​,θ1​)=m1​i=1∑m​(hθ​(x(i))−y(i))x(i)

So then we can repeat the following until convergence:

repeat untill convergence {
θ0:=θ0−αδδθ0J(θ0,θ1)\qquad \theta_0 := \theta_0 - \alpha \deriv{0}θ0​:=θ0​−αδθ0​δ​J(θ0​,θ1​)
θ1:=θ1−αδδθ1J(θ0,θ1)\qquad \theta_1 := \theta_1 - \alpha \deriv{1}θ1​:=θ1​−αδθ1​δ​J(θ0​,θ1​)
} (for jjj = 0 and jjj=1 and simultaneously)

Which is a convex quadratic function for linear regression →\rightarrow→ bow-shaped function that does not have any other optimum besides the global optimum.

"Batch" gradient descent: use all of the training examples →\ra→ we sum over all m training samples.

Answer to the question in the video:

  • Q: Which of the following are true statements? Select all that apply.
  • A:
    • [ ] To make gradient descent converge, we must slowly decrease α\alphaα over time.
    • [ ] Gradient descent is guaranteed to find the global minimum for any function J(θ0,θ1)J(\t{0}, \t{1})J(θ0​,θ1​).
    • [x] Gradient descent can converge even if α\alphaα is kept fixed. (But α\alphaα cannot be too large, or else it may fail to converge.)
    • [x] For the specific choice of cost function J(θ0,θ1)J(\t{0}, \t{1})J(θ0​,θ1​) used in linear regression, there are no local optima (other than the global optimum).

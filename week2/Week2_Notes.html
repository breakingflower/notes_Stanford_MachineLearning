Gradient Descent in practice

Feature scaling

Make sure that the features you are using are on SIMILAR scale.

  • x1x_1x1​ = size (0-5000m)
  • x2x_2x2​ = number of rooms (1-10)

If you try this, the result is an ellipse, and it will take a long time to reach the minimum. In this case, GD will have a much harder time to find the global minimum.

In these settings it's good to scale the features, this is also called feature normalization.

The way to do this is to get every feature into approximately the scale −1≤xi≤1-1 \leq x_i \leq 1−1≤xi​≤1. Approximately means −3-3−3 to +3+3+3 is ok, −13\frac{-1}{3}3−1​ to 13\frac{1}{3}31​ is ok, but -0.000001 to 0.00001 is NOT ok.

Mean Normalization

Replace xix_ixi​ with xi−μix_i - \mu_ixi​−μi​, with μi\mu_iμi​ being the average of the variable (do not apply to x0x_0x0​). For example:

  • x1=size−10002000x_1 = \frac{size - 1000}{2000}x1​=2000size−1000​
  • x2=num_bedrooms−25x_2 = \frac{num\_bedrooms - 2}{5}x2​=5num_bedrooms−2​

More generally, with μi\mu_iμi​=avg value and si=(max−min)s_i = (max-min)si​=(max−min) or stdev:

x1←xi−μisix_1 \leftarrow \frac{x_i - \mu_i}{s_i}x1​←si​xi​−μi​​

NOTE: Quizzes use (max-min) for s_i but programming exercises use stdev!

Learning rate

Assert that GD is working correctly! You can do this by plotting the cost J(θ)J(\t{})J(θ​), it should decrease after every iteration. At some point, it should flatten out and converge.

Declare convergence if J(θ)J(\t{})J(θ​) decreases by less than 10−310^{-3}10−3 in one iteration.

If J(θ)J(\t{})J(θ​) is increasing, GD is not working, you probably need to lower α\alphaα, because it keeps overshooting the minimum of the cost function. And ofcourse make sure you dont have bugs :).

If J(θ)J(\t{})J(θ​) is **going up and down and up and down ... **, also use a smaller α\alphaα.

For sufficiently small α\alphaα, J(θ)J(\t{})J(θ​) should decrease every iteration. If α\alphaα is too small, it may be slow to converge.

To choose α\alphaα, try 0.001,0.003,0.01,0.03,0.1,0.3,1,...0.001, 0.003, 0.01, 0.03, 0.1, 0.3, 1, ...0.001,0.003,0.01,0.03,0.1,0.3,1,... (i.e. 3-fold) and plot J(θ)J(\t{})J(θ​). Pick the one that looks the best.
